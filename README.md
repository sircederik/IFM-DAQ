# IFM-DAQ 🌡️
Sistema de Adquisición de Datos (DAQ) modular para 4 sensores DS18B20 usando Arduino Nano y Python.

## Características
* Lectura en tiempo real de 4 canales.
* Visualización gráfica con Matplotlib.
* Registro automático en CSV con Timestamp.
* Compensación de error (Offsets) por software.

## Requisitos
```bash
pip install pyserial matplotlib
```
